Knockout-foreach-lazy

knockout-foreach-lazy is an extension for the current "foreach" binding that only draws content visible in a scrollable viewport. This should server as a performance improvement for anyone who is rendering a list that is larger than the visible area.

Please check with the GitHub repository for more details.

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
